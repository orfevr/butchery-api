# butchery-api

### Stack
- NodeJS 8.5.0 (docker alpine image)
- Redis 4.0.1 (docker alpine image)
- Docker for Mac 17.06.2

### How to run

You need to have [Docker](https://docs.docker.com/engine/installation/) installed previously.

```
$ bash < rock.sh
```
